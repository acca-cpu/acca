# Acca

Ariel's Custom CPU Architecture

This is my attempt at designing my own toy CPU architecture. Ideally, I'd like
to create:

  * An emulator
  * An assembler
  * A C compiler backend (probably for LLVM)
  * Possibly a JIT recompiler (maybe based on LLVM)
